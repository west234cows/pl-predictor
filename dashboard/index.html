🔧 Connect to Supabase

Enter your Supabase project URL and anon (public) key. These are stored only in your browser's local storage and never sent anywhere else. Find them in your Supabase dashboard under Project Settings → API.

ℹ️ About this app

Predictions are generated by a Poisson Monte Carlo model (100,000 simulations) using rolling expected-goals (xG) data from Understat. The pipeline runs automatically via GitHub Actions every Thursday and Monday — no server required.

Model: v1.1-poisson-dc (Dixon-Coles low-score correction applied).

Data: Understat · Powered by Supabase + GitHub Actions
